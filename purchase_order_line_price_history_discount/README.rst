This module extends the functionality of 'Purchase order line price history'
module to add a discount column to the list of prices in the wizard.

**Table of contents**

.. contents::
   :local:

Usage
=====

#. Go to *Purchase -> Requests for Quotation* and select
   a Requests for Quotation or create a new one.
#. Click on the clock button in one of the order lines and you will
   see now a new column in the list of prices in the wizard.
