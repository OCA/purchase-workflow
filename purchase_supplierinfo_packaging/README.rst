This module extends the functionality of product_supplierinfo_packaging
(product-attribute) to support supplier info packaging and to allow you
to assign the packaging on the purchase order line depending on the
supplier info.

**Table of contents**

.. contents::
   :local:

Use Cases / Context
===================

When using packaging on supplier info, you might want to assign
automatically the packaging on the purchase order line when creating it.

Usage
=====

To use this module, you need to:

1. Use packaging
2. Assign a packaging on a product supplier info
3. Create a purchase order linked to the supplier info partner
4. Use the previous product, the packaging will be assigned
   automatically
